第1問
分野: ストラテジ系 / 知的財産権・生成AI

問題: 生成AIを用いた生成物の取扱いに関して、既存の著作物の著作権者から許諾を得ることが必要となる可能性のあるものだけを、全て挙げたものはどれか。

a: 好みのアーティストの楽曲に似た音楽が得られるように生成AIを用いて楽曲を生成し、その楽曲をインターネット上にアップロードし、無料で公開した。

b: 好みのアーティストの楽曲に似た音楽が得られるように生成AIを用いて楽曲を生成し、その楽曲を自分のPC上に保管し、個人で視聴した。

c: 生成AIで音楽を生成したところ、偶然好みのアーティストの楽曲に似た音楽が生成できたので、自分のPC上に保管し、個人で視聴した。

選択肢:

ア: a

イ: a, b

ウ: a, b, c

エ: b, c

正解: ア

解説: aは、好みのアーティストに「似た」楽曲を意図的に生成し、それをインターネット上にアップロード（公衆送信）して無料公開しているため、著作権侵害（翻案権や公衆送信権などの侵害）にあたり、既存の著作物の著作権者から許諾を得ることが必要となる可能性が極めて高いです。一方、bおよびcは「個人で視聴（私的使用のための複製）」の範囲内にとどまるため、著作権法上、原則として許諾は不要です。したがって、許諾が必要となる可能性のあるものは「a」のみです。

出典: 令和8年度 ITパスポート試験 問1

第2問
分野: ストラテジ系 / 知的財産権・ビジネスモデル特許

問題: インターネットを利用した企業広告に関する新たなビジネスモデルを知的財産として出願し、コンピュータシステムとして実現した。このビジネスモデルを知的財産として、保護する法律はどれか。

選択肢:

ア: 意匠法

イ: 実用新案法

ウ: 著作権法

エ: 特許法

正解: エ

解説: コンピュータやインターネットなどのICT技術を用いて具体的に実現されたビジネス手法（ビジネスモデル）は、「発明」として特許法の保護対象（ビジネスモデル特許）になり得ます。意匠法は物品等の外観デザインを保護し、実用新案法は物品の形状や構造などの考案を保護し、著作権法は思想や感情を創作的に表現した著作物（表現そのもの）を保護します。ビジネスモデルというアイデアや仕組みそのものを直接保護するものではないため、特許法が最も適切です。

出典: 令和8年度 ITパスポート試験 問5

第3問
分野: ストラテジ系 / 法的遵守・個人情報保護

問題: 個人情報を法令に定める方法で匿名化し、復元できないようにしたもので、一定のルールの下で本人の同意なく利活用できる情報を表す用語として、最も適切なものはどれか。

選択肢:

ア: 暗号化情報

イ: 個人識別符号

ウ: 匿名加工情報

エ: 要配慮個人情報

正解: ウ

解説: 個人情報保護法において、特定の個人を識別できないように個人情報を加工し、かつ、当該個人情報を復元できないようにした情報を「匿名加工情報」と呼びます。匿名加工情報は、一定のルールの下で本人の同意を得ることなく、第三者への提供や利活用が可能となります。

第4問
分野: ストラテジ系 / 企業活動・国際規格

問題: ISO 26000とは、組織の社会的責任についての国際規格である。ISO 26000で定められている、企業が果たすべき社会的責任を表す用語として、最も適切なものはどれか。

選択肢:

ア: CSR

イ: SDGs

ウ: コーポレートガバナンス

エ: ソーシャルメディアポリシー

正解: ア

解説: ISO 26000は「社会的責任（Social Responsibility）」に関する国際的なガイドライン（規格）です。企業が果たすべき社会的責任を意味する用語は「CSR（Corporate Social Responsibility）」です。SDGsは持続可能な開発目標、コーポレートガバナンスは企業統治、ソーシャルメディアポリシーはSNS利用のガイドラインであり、社会的責任そのものを表す用語ではありません。

第5問
分野: ストラテジ系 / 法的遵守・セキュリティ関連法

問題: サイバーセキュリティ基本法の立法趣旨を説明したものとして、最も適切なものはどれか。

選択肢:

ア: コンピュータへの不正アクセス行為を定義し、その行為を禁止するとともに罰則を定め、ネットワーク通信秩序の維持を図ること

イ: 情報セキュリティに関する適切なコントロールを整備、運用するための実践的な規範を定め、企業などで効果的な情報セキュリティ対策の推進を促すこと

ウ: 情報通信ネットワークを通じたコンピュータへの不正侵入などの行為に対する防御施策に関し、基本理念を定め、国及び地方公共団体の責務などを明らかにすること

エ: デジタルコンテンツの複製防止のための技術的制限手段を無効化する行為などを規制することによって、企業などの営業上の利益を確保すること

正解: ウ

解説: サイバーセキュリティ基本法は、サイバーセキュリティに関する施策について、基本理念を定め、国、地方公共団体、重要インフラ事業者などの責務を明らかにするとともに、国の基本方針となる「サイバーセキュリティ戦略」の策定などを規定した法律です。アは「不正アクセス禁止法」、イは「情報セキュリティ管理基準」などの説明、エは「不正競争防止法」や「著作権法」の技術的保護手段の回避規制に関する記述です。

第6問
分野: ストラテジ系 / 業務改善・RPA

問題: 伝票入力処理などの定型的な事務作業を、ソフトウェアロボットに代替させることによって、自動化や効率化を図る手段として、最も適切なものはどれか。

選択肢:

ア: BPO

イ: FA

ウ: RPA

エ: SFA

正解: ウ

解説: RPA（Robotic Process Automation）は、PC上で行う定型的な事務作業（データの入力や転記など）を、ソフトウェアで構築されたロボットに代行させることで、自動化・効率化を図る技術・手段です。BPOは業務プロセスのアウトソーシング、FAは工場（生産ライン）の自動化（主に物理ロボット等を使用）、SFAは営業支援システムです。

第7問
分野: ストラテジ系 / 業務改善・リーン生産方式

問題: リーン生産方式による業務改善の観点である「ムリ、ムラ、ムダ」のうち、最も「ムダ」に当たるものとして、適切なものはどれか。

選択肢:

ア: 属人化していて、担当者によって作成物の品質が異なる作業

イ: 組織の能力以上に負荷をかける無謀な計画や作業標準

ウ: 待機や必要以上の加工など、付加価値を生まない動作

エ: 長時間、身体の一部に負担がかかり、健康を害するおそれのある作業

正解: ウ

解説: リーン生産方式における「ムダ」とは、付加価値を生まない作業やプロセスのことです。代表的な例として「手待ち（待機）」や「過剰な加工（必要以上の加工）」などがあります。アの品質のばらつきは「ムラ」、イの能力以上の負荷は「ムリ」、エの身体への負担も「ムリ」に該当します。

第8問
分野: ストラテジ系 / 経営戦略・SCM

問題: 購買、製造、販売という供給者（サプライヤー）から消費者までを結ぶ一連の業務のつながりを総合的に管理し、資材の調達から顧客への販売に至る在庫などの無駄をなくして、プロセス全体の最適化を図るものはどれか。

選択肢:

ア: CRM

イ: ERP

ウ: SCM

エ: SFA

正解: ウ

解説: 供給者（サプライヤー）から消費者（顧客）までを結ぶ一連の調達・生産・物流・販売の流れを「サプライチェーン」と呼び、これを総合的に管理してプロセス全体の最適化や在庫削減を図る経営手法を「SCM（Supply Chain Management）」と言います。CRMは顧客関係管理、ERPは企業資源計画（基幹業務統合）、SFAは営業活動支援です。

第9問
分野: マネジメント系 / プロジェクトマネジメント・倫理・契約

問題: ソフトウェア導入プロジェクトの作業に関する記述のうち、適切な行動（倫理や契約面における判断）をとっているものだけを全て挙げたものはどれか。

a: 導入作業中に契約外のPCが見つかった。そのPCを導入対象外とした。

b: 導入作業中に契約外のPCへの導入を依頼された。依頼に基づいて、その対応を無償で行った。

c: 導入作業の続行が困難な状態となった。発注者と合意し、導入前の状態に復旧した。

選択肢:

ア: a, b

イ: a, c

ウ: b

エ: b, c

正解: イ

解説: aは、契約範囲外のPCは作業対象外とするのが契約上適切です。bは、契約外の作業を無償で安易に行うことはプロジェクトの範囲（スコープ）逸脱やトラブルの原因となるため、適切ではありません（有償での追加契約などの手続きが必要です）。cは、続行不可能なトラブル発生時に発注者と合意（合意形成）の上で、作業前の状態にロールバック（復旧）しているため、適切なリスク対応行動です。したがって、適切な行動は「a」と「c」です。

第10問
分野: テクノロジ系 / セキュリティ要素

問題: ある企業では、サーバ室への入室管理を暗証番号で行っていたが、虹彩認証装置による本人確認を加えることによって、無許可の者の入室をより確実に防止する対策とした。これは、サーバ室における情報セキュリティ要素のうち、どれを高める対策に該当するか。

選択肢:

ア: 可用性

イ: 完全性

ウ: 機密性

エ: 否認防止

正解: ウ

解説: 無許可の者のサーバ室入室（不正侵入）を防ぎ、アクセスを許可された者だけに制限する対策は、情報の「機密性（Confidentiality）」を高めることに直接寄与します。可用性は必要な時に使える状態の維持、完全性はデータが改ざんされない状態の維持、否認防止は行動の事後証明に関わる要素です。
